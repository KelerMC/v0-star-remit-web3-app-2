# 🚀 EcoXport - Prototipo para Ideathon

## 🎯 Sobre el Proyecto

**EcoXport** es un marketplace blockchain que conecta **productores peruanos** con **compradores internacionales**, eliminando intermediarios y usando la red Stellar para pagos instantáneos.

---

## ✨ Features Innovadoras del Prototipo

### 1. **Dual Role System** 🎭
- Vista Producer (Productor/Exportador)
- Vista Buyer (Comprador/Importador)
- Cambio de rol en tiempo real

### 2. **Smart Escrow Automático** 🛡️
- Pagos protegidos en blockchain
- Liberación automática con tracking
- Visualización del proceso en tiempo real

### 3. **Certificados como NFTs** 🏆
- USDA Organic, GlobalGAP, Fair Trade
- Verificables en blockchain
- Inmutables y portables

### 4. **AI Analytics Dashboard** 🤖
- Predicciones de ventas
- Recomendaciones de inversión
- Demanda forecasting

### 5. **Documentos Automáticos** 📄
- Contratos firmados digitalmente
- Facturas SUNAT electrónicas
- Exportación en un click

## 🚀 Cómo Correr el Prototipo

### **Instalación:**

\`\`\`bash
# Instalar dependencias
npm install

# Correr en desarrollo
npm run dev

# Abrir en navegador
http://localhost:3000
\`\`\`

### **Navegación:**

1. **Landing:** Selector de roles (Producer/Buyer)
2. **Producer View:**
   - Dashboard con stats
   - AI Predictions
   - Órdenes recientes
   - Generación de documentos

3. **Buyer View:**
   - Marketplace con productos
   - Certificados verificados
   - Dashboard con órdenes
   - Smart Escrow activo

4. **Cambiar rol:** Click en "Cambiar rol" en navbar
